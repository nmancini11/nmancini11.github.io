Skatesense

PRECOURSE

When you look at action sports from a statistical perspective, you quickly realize how little data is being captured. I set out on a mission to quantify competitive skateboarding by capturing the data for every trick. This all started with my first analysis on The Berrics Run + Gun competition.
Over the last few months I have built a rails application for logging trick data from professional skateboarding competitions.

METRICS

By logging the timestamp, stance, and trick we can derive some interesting metrics exclusive to skateboarding:

  • Consecutive Flip Tricks
  • Breakdown of stance: Regular | Nollie | Fakie | Switch
  • Consecutive combinations
  • Flip-In + Flip-Out

Albeit this does not paint a full picture since the obstacle the trick was done on varies greatly in difficulty throughout the park; you can start see how consistency and the caliber of tricks shapes a skaters performance.
